# Политика Безопасности

Мы серьезно относимся к безопасности проекта и ценим помощь сообщества в поиске уязвимостей. Если вы обнаружили проблему безопасности, пожалуйста, следуйте инструкциям ниже для её конфиденциального сообщения.

## Поддерживаемые Версии

В настоящее время обновления безопасности выпускаются только для актуальной версии в ветке `main` и последней версии проекта в ветке `vX.X.X`, а также для открытых PR.

| Версия | Поддержка Безопасности |
| :--- | :--- |
| Актуальная ветка (`main`) | :white_check_mark: Поддерживается |
| Ветка последней версии (`vX.X.X`) | :white_check_mark: Поддерживается |
| Открытые/Draft/Merged Pull Request | :white_check_mark: Поддерживается |
| Закрытые без Merge Pull Request | :white_check_mark: Не поддерживаются |
| Старые версии/архивы | :x: Не поддерживаются |

## Как Сообщить Об Уязвимости

> [!IMPORTANT]
> **Пожалуйста, не публикуйте информацию об уязвимостях в открытом доступе (в GitHub Issues, Pull Requests, публичных каналах Discord и т.д.).** Это поможет защитить пользователей бота и данные проекта до тех пор, пока не будет выпущено исправление.

Чтобы Сообщить Об Уязвимости:

1. Перейдите на наш Discord-сервер: **[discord.gg/MoscowRus](https://discord.gg/MoscowRus)**.
2. Создайте приватный тикет (обращение в поддержку).
3. В тикете укажите, что вы хотите сообщить о наличии уязвимости в безопасности бота/когов.
4. После ответа разработчиков в тикете, один из ответственных разработчиков свяжется с вами в личных сообщениях для уточнения технических деталей и воспроизведения уязвимости.

Мы постараемся оперативно обработать ваше обращение и выпустить исправление. Спасибо за ваш вклад в безопасность сообщества!
